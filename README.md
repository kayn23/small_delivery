# Акторы

## Клиент

- Посмотреть список своих заказов, фильтр:
  - Отправитель
  - Получатель
  - В процессе
  - Законченные
- Создать заявку на новую доставку

## Администратор

- Поиск по заказам
- Создавать и изменять заказы
- Изменять статус заказов
- Добавлять и изменять склады
- И весь функционал клиента
